## 1. 项目概述

### 1.1 项目背景

SoundBox 是一款专为**音声作品（ASMR/广播剧）**设计的桌面端高保真音频播放器。主要解决以下问题：

- DLsite Play 等在线播放器依赖网络，无法播放本地文件
- 通用播放器缺乏对音声作品特性的支持（如字幕同步、章节管理）

### 1.2 目标用户

- 音声作品爱好者
- 对音质有较高要求的用户
- 拥有本地音声文件收藏的用户

### 1.3 核心价值

- **专为音声作品优化** - 字幕同步、章节管理、无缝播放
- **本地播放** - 无需网络，隐私保护

## 3. 音频解析需求（核心）

### 3.1 音频规格支持

### 3.2 支持的音频格式

| 格式 | 压缩类型 | 优先级 | 说明 |
|------|----------|--------|------|
| WAV | 无压缩 | P0 | 音声作品最常见格式 |
| FLAC | 无损压缩 | P0 | 开源无损格式 |
| AIFF | 无压缩 | P1 | Apple 无损格式 |
| ALAC | 无损压缩 | P1 | Apple 无损压缩 |
| MP3 | 有损压缩 | P0 | 兼容性 |
| AAC | 有损压缩 | P1 | 兼容性 |
| OGG | 有损压缩 | P2 | 开源有损格式 |
| DSD (DSF/DFF) | DSD | P3 | 发烧级格式 |
| MQA | 高效编码 | P3 | 需要解码器支持 |

### 3.3 解码器选择

| 库 | 语言 | 支持格式 | 特点 |
|---|------|---------|------|
| **FFmpeg (libavcodec)** | C | 全格式 | 工业标准，最全面 |
| **libFLAC** | C | FLAC | 官方参考实现 |
| **libsndfile** | C | WAV/AIFF/FLAC | 简洁易用 |
| **miniaudio** | C | 全格式 | 单头文件，轻量 |
| **Symphonia** | Rust | 全格式 | Rust 原生，跨平台 |
| **rodio** | Rust | 基础格式 | Rust 高级封装 |

## 5. 数据模型

### 5.1 核心模型

```swift
// 音频格式
struct AudioFormat {
    var sampleRate: Double      // 采样率 (44100, 96000, 192000...)
    var bitDepth: Int           // 位深度 (16, 24, 32)
    var channels: Int           // 声道数 (1, 2)
    var isFloat: Bool           // 是否浮点
    var isBigEndian: Bool       // 字节序
}

// 音频文件
struct AudioFile {
    let url: URL
    let name: String
    let format: AudioFormat
    let duration: TimeInterval
    let fileSize: UInt64
    let subtitleURL: URL?
}

// 音轨
struct Track {
    let audioFile: AudioFile
    var index: Int
    var title: String
    var artist: String?
    var album: String?
}

// 播放列表
class Playlist {
    var tracks: [Track]
    var currentIndex: Int
    var repeatMode: RepeatMode
    var isShuffled: Bool
}

// 播放状态
class PlayerState {
    var playbackState: PlaybackState
    var currentTime: TimeInterval
    var totalDuration: TimeInterval
    var volume: Float
    var playbackRate: Float
    var currentSubtitle: String?
}
```

### 6.2 设计原则

- **简洁专注** - 减少视觉干扰，专注内容
- **信息层次** - 格式信息、Hi-Res 标识清晰可见
- **响应式** - 支持窗口缩放
- **深色模式** - 默认深色，护眼设计
